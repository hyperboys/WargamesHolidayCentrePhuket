1. Overview

The Events Management system allows administrators to create, edit, and manage wargaming events that appear on the public website. Each event contains:

  • Bilingual content (English and Thai)
  • Date range (start and end dates)
  • Pricing information
  • Player capacity
  • Featured image
  • Event highlights/inclusions
  • Historical background
  • Game rules

Image Best Practices

DO:

  • Show miniatures in action
  • Include terrain/scenery
  • Good lighting & focus
  • Landscape orientation
  • Compress before upload

DON'T:

  • Use low-resolution images
  • Use copyrighted photos
  • Include people's faces
  • Use portrait orientation
  • Upload files over 5MB

9. Deleting Events

Warning: Deletion is Permanent!
Deleted events cannot be recovered. Consider changing status to "Past" instead of deleting.

How to Delete

  1. Step 1: Find the event to delete
  2. Step 2: Click the Delete button
  3. Step 3: Review the confirmation dialog
  4. Step 4: Click Delete to confirm or Cancel to abort

10. Best Practices

Content Guidelines

ElementBest Practice
TitleClear, descriptive (e.g., "Waterloo Campaign 2026")
SlugShort, lowercase, hyphens only (e.g., "waterloo-2026")
Description2-3 paragraphs, highlight unique features
HistoryEducational content about the battle/era
Highlights5-8 bullet points of key inclusions

Workflow Tips

1. Create as Draft First

  • Fill in all content
  • Review for accuracy
  • Have someone else proofread

2. Test Before Going Active

  • Set to "Upcoming" first
  • Check public website display
  • Verify image loads correctly

3. Keep Events Updated

  • Update status as dates approach
  • Archive past events promptly
  • Remove outdated information

IssueSolution
Required fields emptyFill in Title (EN), Dates, and Price

Getting Help

If problems persist:

  1. Check browser console (F12 - Console tab) for errors
  2. Take screenshots of the issue
  3. Note the steps that caused the problem
  4. Contact technical support with details
